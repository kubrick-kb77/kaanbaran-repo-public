## Contributing

Contribution is welcome, and should follow the guidelines described here:

https://www.terraform-best-practices.com/

### Prerequisites

* Visual Studio Code or Codium
* [`terraform-docs`](https://terraform-docs.io/)
* [`pre-commit`](https://pre-commit.com/)

### Setup

Install the pre-commit hooks, so any commit also updates [README.md](../README.md).

```
pre-commit
```
